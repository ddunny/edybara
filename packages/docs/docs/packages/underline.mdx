# Underline

에디터에서 **밑줄** 텍스트를 표현하기 위한 서식과 관련 플러그인을 제공합니다.

## 설치

```bash
npm install @edybara-editor/underline
```

## 스타일

```css title="src/index.scss"
@import "@edybara-editor/underline/styles/underline.scss";
```

## 서식

#### `edybaraUnderlineMarks`

밑줄 텍스트 서식을 제공합니다.

## 플러그인

#### `edybaraUnderlinePlugins`

하단의 모든 플러그인을 통합 제공합니다.

#### `edybaraUnderlineKeymapPlugins`

현재 텍스트를 밑줄 텍스트로 전환하는 단축키(Mod-u)를 제공합니다.

## 사용 예시

```typescript title="src/index.ts"
import { EditorState, Plugin } from 'prosemirror-state';
import { Schema } from 'prosemirror-model';
import { EditorView } from 'prosemirror-view';
import { edybaraBaseNodes, edybaraCorePlugins } from '@edybara-editor/core';
import { edybaraParagraphNodes, edybaraParagraphPlugins, } from '@edybara-editor/paragraph';
import { edybaraMenubarPlugins } from '@edybara-editor/menubar';
import { edybaraUnderlineMarks, edybaraUnderlinePlugins } from '@edybara-editor/underline';

const schema = new Schema({
  nodes: {
    ...edybaraBaseNodes(),
    ...edybaraParagraphNodes(),
  },
  marks: {
    ...edybaraUnderlineMarks({
      markName: 'underline',
    }),
  },
});

const plugins: Plugin[] = [
  ...edybaraParagraphPlugins({
    nodeType: schema.nodes['paragraph'],
  }),
  ...edybaraUnderlinePlugins({
    markType: schema.marks['underline'],
  }),
  ...edybaraMenubarPlugins({
    textStyles: {
      underlineMarkType: schema.marks['underline'],
    },
  }),
  ...edybaraCorePlugins(),
];

const view = new EditorView(document.querySelector('#editor'), {
  state: EditorState.create({
    schema: schema,
    plugins: plugins,
  }),
});
```
